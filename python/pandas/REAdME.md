# Pandas
  
Proporciona estructuras de datos y funciones de alto nivel que nos permiten trabajar con datos estructurados. Estas estructuras y funciones son, normalmente de las más usadas en análisis de datos.  
Los objetos ofrecidos por pandas son el *DataFrame* que es una estructura tabular bidimensional y la *serie*.  
Para poder utilizar la libreria de pandas es necesario importar la libreria correspondiente:  
  
`import pandas as pd`.
